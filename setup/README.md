# Setup

## System requirements

If you are building and running Chaste on your own computer, the specifications below serve as a guide on the resources required.

|                 | Minimum           | Recommended       |
| --------------- | ----------------- | ----------------- |
| Memory          | 8 GB              | 16 GB             |
| Free Disk Space | 15 GB             | 30 GB             |
| CPU             | 2 cores @ 1.5 GHz | 4 cores @ 2.0 GHz |
